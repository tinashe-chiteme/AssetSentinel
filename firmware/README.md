# Firmware

The firmware for AssetSentinel is currently undergoing active development.

The project began as VibeCheck during the Huawei Code4Mzansi competition before evolving into AssetSentinel.

Future firmware releases may include:

- Improved modularity
- Enhanced vibration classification
- Calibration routines
- Improved fault handling
- Cloud integration
- Support for multiple devices

The firmware published in this repository represents educational prototype development rather than production-ready software.
